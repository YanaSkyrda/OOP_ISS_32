# Task 4

### Вывод описания класса

Используя механизм рефлексии, напишите программу, которая выводит на экран полное описание класса с заданным именем, включающее всю информацию о классе за исключением инструкций импорта, комментариев и кода инициализаторов, конструкторов и методов. Для загрузки класса использовать собственный загрузчик.
